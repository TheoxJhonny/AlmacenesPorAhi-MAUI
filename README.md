# Almacenes Por Ahí

Sistema de gestión de inventario desarrollado en .NET MAUI.

## Características

- Gestión de productos
- Gestión de clientes
- Registro de ventas
- Registro de devoluciones
- Actualización de stock
- Arquitectura MVVM
- ObservableCollection
- INotifyPropertyChanged

## Tecnologías

- .NET MAUI
- C#
- XAML
- MVVM
